## Development Environment

There are no restrictions on the development environment and editor. However, you have to specify the build environment and package.

| Operation and Build environment | Version  |
| ------------------------------- | -------- |
| Node.js                         | ^14.18.1 |
| npm                             | ^6.14.15 |

| Package    | Version |
| ---------- | ------- |
| React      | ^17.0.2 |
| TypeScript | ^4.4.4  |

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build production
$ npm run build
```
